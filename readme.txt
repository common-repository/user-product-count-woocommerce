User Product Count WooCommerce - adds custom column of User Product count in wordpress users list page

== Description ==

User Product Count WooCommerce - adds custom column of User Product count in wordpress users list page

Major features in User Product Count WooCommerce include:

* You can see total products of each user in Multi vendor/seller woocommerce site
* This plugin add one custom column on users list "User Product"

== Installation ==

Upload the User Product Count WooCommerce plugin to your woocommerce site, Activate it, then check it on users list page.

== Frequently Asked Questions ==

= Can I use this plugin without WooCommerce plugin? =

No, You must have to install woocommerce plugin before use this plugin.
